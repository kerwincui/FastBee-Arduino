# 扩展实验19：RFID射频卡（MFRC522）

## 实验概述

MFRC522 是一款常用的 RFID 读卡器，支持读取 Mifare 系列 IC 卡（13.56MHz）。通过 SPI 接口通信，可实现门禁、考勤、物品标记等应用。FastBee 在 ESP32-S3（full 固件）中内置 RFID 驱动。

> **注意**：此功能仅在 ESP32-S3（full 固件）中可用，需要 `FASTBEE_ENABLE_RFID=1`。

## 硬件接线

| 开发板标识 | GPIO引脚 | 连接设备 |
|-----------|---------|---------|
| SDA(SS) | GPIO5 | MFRC522 SDA（片选） |
| SCK | GPIO18 | MFRC522 SCK |
| MOSI | GPIO23 | MFRC522 MOSI |
| MISO | GPIO19 | MFRC522 MISO |
| RST | GPIO27 | MFRC522 RST（复位） |

> MFRC522 供电 3.3V，不可接 5V！

### 方式1：Web界面配置（推荐）

#### 步骤1：进入外设管理页面

1. 打开浏览器访问 ESP32 IP 地址
2. 登录后点击左侧菜单 **外设配置**

#### 步骤2：添加RFID读卡器

1. 点击 **<i class="fas fa-plus"></i> 新增外设** 按钮
2. 填写配置：

   | 字段 | 填写内容 | 说明 |
   |------|---------|------|
   | **外设ID** | `rfid_01` | RFID读卡器 |
   | **名称** | `RFID读卡器` | 显示名称 |
   | **外设类型** | **通用传感器** (type: 38) | SPI接口 |
   | **引脚配置** | `5,18,23,19,27` | SS,SCK,MOSI,MISO,RST（顺序重要） |
   | **传感器类别** | `rfid` | RFID驱动 |

3. 点击 **保存**

> ⚠️ **注意**：MFRC522必须使用3.3V供电，5V会损坏芯片！

## 外设执行联动

### 场晦1：刷卡开锁（事件触发）

**功能**：检测到RFID卡片时，打开继电器3秒后关闭

#### Web界面配置步骤

**步骤1：确保已配置继电器外设**

- 外设ID：`relay_01`
- 类型：GPIO输出

**步骤2：创建规则**

1. 点击左侧菜单 **外设配置** → 切换到 **外设执行管理** 标签
2. 点击 **<i class="fas fa-plus"></i> 新增规则** 按钮
3. 填写基础配置：
   - **规则名称**：`RFID开锁`
   - **上报数据**：✅ 启用
   - **启用**：✅ 启用

**步骤3：配置触发器（事件触发）**

1. 点击 **添加触发** 按钮
2. 填写触发器配置：

   | 字段 | 填写内容 | 说明 |
   |------|---------|------|
   | **触发类型** | 选择 **事件触发** | RFID刷卡事件 |
   | **事件类型** | `120` | RFID卡检测到 |
   | **事件来源** | 选择 `rfid_01` | RFID读卡器 |

**步骤4：配置动作（需要3个动作）**

**动作1：打开继电器**

1. 点击 **添加动作** 按钮
2. 填写：
   - **动作类型**：选择 **高电平**
   - **目标外设**：选择 `relay_01`

**动作2：延时3秒**

1. 再次点击 **添加动作** 按钮
2. 填写：
   - **动作类型**：选择 **延时**
   - **延时时间**：`3000`（3000ms=3秒）

**动作3：关闭继电器**

1. 第三次点击 **添加动作** 按钮
2. 填写：
   - **动作类型**：选择 **低电平**
   - **目标外设**：选择 `relay_01`

3. 点击 **保存** 按钮

### 场晦2：卡片移除事件（事件触发）

**功能**：RFID卡片移除时关闭LED

**配置步骤**：

1. 创建规则，名称：`RFID卡移除`
2. 触发器配置：
   - **触发类型**：选择 **事件触发**
   - **事件类型**：`121`（RFID卡移除）
   - **事件来源**：选择 `rfid_01`
3. 动作配置：
   - **动作类型**：选择 **高电平**（LED熄灭）
   - **目标外设**：选择 `led_d1`
4. 点击 **保存**

### RFID说明

| 特性 | 说明 |
|------|------|
| 工作频率 | 13.56MHz |
| 支持卡片 | Mifare系列IC卡 |
| 读取距离 | 3-5cm |
| UID格式 | 4字节十六进制（如A1B2C3D4） |
| 轮询频率 | 内部200ms轮询一次 |

## 注意事项

1. **仅 S3-full**：需要 ESP32-S3 + full 固件
2. **供电**：MFRC522 必须使用 3.3V，5V 会损坏芯片
3. **读取距离**：典型读卡距离 3-5cm
4. **UID 格式**：卡片 UID 为 4 字节十六进制，如 `A1B2C3D4`
5. **SPI 引脚**：引脚顺序为 SS, SCK, MOSI, MISO, RST
6. **轮询频率**：驱动内部 200ms 轮询一次卡片
