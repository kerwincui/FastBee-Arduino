# 实验20：超声波测距（HC-SR04）

## 实验概述

HC-SR04 超声波传感器通过发射和接收超声波来测量距离，测量范围 2~400cm，精度约 3mm。FastBee 内置 `SENSOR` 外设类型（type: 38），配合 `category: ultrasonic` 支持超声波测距。

## 硬件接线

| 开发板标识 | GPIO引脚 | 连接设备 |
|-----------|---------|---------|
| TRIG | GPIO14 | HC-SR04 触发引脚 |
| ECHO | GPIO27 | HC-SR04 回响引脚 |

> TRIG 发送 10μs 高电平脉冲触发测量，ECHO 返回高电平持续时间表示距离。

## FastBee 外设配置

### 方式1：Web界面配置（推荐）

#### 步骤1：进入外设管理页面

1. 打开浏览器访问 ESP32 IP 地址
2. 登录后点击左侧菜单 **外设配置**

#### 步骤2：添加超声波传感器

1. 点击 **<i class="fas fa-plus"></i> 新增外设** 按钮
2. 填写配置：

   **基本信息（左列）：**
   
   | 字段 | 填写内容 | 说明 |
   |------|---------|------|
   | **外设ID** | `ultrasonic_01` | 超声波测距 |
   | **名称** | `超声波测距` | 显示名称 |
   | **外设类型** | 选择 **传感器** (type: 38) | 通用传感器 |
   | **启用** | ✅ 开启 | 默认启用 |

   **引脚和参数（右列）：**
   
   | 字段 | 填写内容 | 说明 |
   |------|---------|------|
   | **引脚配置** | `14,27` | TRIG=GPIO14, ECHO=GPIO27（顺序重要） |
   | **传感器类别** | `ultrasonic` | 超声波测距 |

3. 点击 **保存**

> ⚠️ **注意**：引脚顺序必须是TRIG在前，ECHO在后。ECHO输出5V需分压到3.3V

#### 步骤3：验证配置

1. 在外设列表中查看新增的超声波传感器
2. 确认状态为 **运行中**
3. 点击 **读取** 按钮测试距离值

## 外设执行联动

### 场景1：定时测距（定时触发）

**功能**：每1秒读取一次超声波距离值

#### Web界面配置步骤

**步骤1：创建规则**

1. 点击左侧菜单 **外设配置** → 切换到 **外设执行管理** 标签
2. 点击 **<i class="fas fa-plus"></i> 新增规则** 按钮
3. 填写基础配置：
   - **规则名称**：`超声波定时测距`
   - **上报数据**：✅ 启用
   - **启用**：✅ 启用

**步骤2：配置触发器**

1. 点击 **添加触发** 按钮
2. 填写触发器配置：

   | 字段 | 填写内容 | 说明 |
   |------|---------|------|
   | **触发类型** | 选择 **定时触发** | 定时采集 |
   | **定时模式** | 选择 **固定间隔** | 按毫秒间隔 |
   | **间隔时间** | `1000` | 1秒 |

**步骤3：配置动作**

1. 点击 **添加动作** 按钮
2. 填写：
   - **动作类型**：选择 **读取传感器**
   - **目标外设**：选择 `ultrasonic_01`
   - **数据字段**：`distance`

3. 点击 **保存** 按钮

### 场景2：近距离报警（轮询触发）

**功能**：每0.5秒检测距离，当距离小于20cm时触发蜂鸣器短鸣报警

#### Web界面配置步骤

**步骤1：确保已配置蜂鸣器外设**

- 外设ID：`buzzer_gpio`
- 类型：GPIO输出

**步骤2：创建规则**

1. 点击 **<i class="fas fa-plus"></i> 新增规则**
2. 填写基础配置：
   - **规则名称**：`近距离报警`
   - **上报数据**：✅ 启用
   - **启用**：✅ 启用

**步骤3：配置触发器（轮询触发）**

1. 点击 **添加触发** 按钮
2. 填写触发器配置：

   | 字段 | 填写内容 | 说明 |
   |------|---------|------|
   | **触发类型** | 选择 **轮询触发** | 定时检测条件 |
   | **目标外设** | 选择 `ultrasonic_01` | 超声波传感器 |
   | **轮询间隔** | `500` | 0.5秒 |
   | **条件表达式** | `distance < 20` | 距离小于20cm |

**步骤4：配置动作（需要3个动作）**

**动作1：蜂鸣器响**

1. 点击 **添加动作** 按钮
2. 填写：
   - **动作类型**：选择 **高电平**
   - **目标外设**：选择 `buzzer_gpio`

**动作2：延时200ms**

1. 再次点击 **添加动作** 按钮
2. 填写：
   - **动作类型**：选择 **延时**
   - **动作参数**：`200`

**动作3：蜂鸣器停**

1. 第三次点击 **添加动作** 按钮
2. 填写：
   - **动作类型**：选择 **低电平**
   - **目标外设**：选择 `buzzer_gpio`

3. 点击 **保存** 按钮

### HC-SR04特性说明

| 特性 | 参数 | 说明 |
|------|------|------|
| 测量范围 | 2cm ~ 400cm | 有效测量距离 |
| 精度 | ±3mm | 高精度 |
| 盲区 | 2cm | 最小测量距离 |
| 测量角度 | 15° | 声波锥形发射角 |
| 采样间隔 | ≥60ms | 避免回波干扰 |

## 注意事项

1. **供电**：HC-SR04 需要 5V 供电，ECHO 输出为 5V，需分压到 3.3V 再接 ESP32
2. **盲区**：最小测量距离 2cm，低于此距离不可靠
3. **测量角度**：声波锥形发射角约 15°，狭窄空间可能有反射干扰
4. **温度补偿**：声速随温度变化（20°C约343m/s），精确测量需补偿
5. **采样频率**：建议不低于 60ms 间隔，避免前后回波干扰
6. **引脚顺序**：pins 数组第一个为 TRIG，第二个为 ECHO
