# 扩展实验5：对射光电传感器

## 实验概述

对射式光电传感器由发射管和接收管组成，当光路被遮挡时输出信号变化。适用于计数、限位检测、传送带物品检测等工业场景。

## 硬件接线

| 开发板标识 | GPIO引脚 | 连接设备 |
|-----------|---------|---------|
| OUT | GPIO4 | 光电传感器 DO输出（遮挡时低电平） |

## FastBee 外设配置

### 方式1：Web界面配置（推荐）

#### 步骤1：进入外设管理页面

1. 打开浏览器访问 ESP32 IP 地址
2. 登录后点击左侧菜单 **外设配置**

#### 步骤2：添加对射光电传感器

1. 点击 **<i class="fas fa-plus"></i> 新增外设** 按钮
2. 填写配置：

   | 字段 | 填写内容 | 说明 |
   |------|---------|------|
   | **外设ID** | `photoelectric_01` | 对射光电 |
   | **名称** | `对射光电` | 显示名称 |
   | **外设类型** | **数字输入上拉** (type: 13) | 内部上拉 |
   | **引脚配置** | `4` | OUT对应GPIO4 |

3. 点击 **保存**

> 💡 **提示**：光路通畅时高电平，被遮挡时低电平

## 外设执行联动

### 场景：物品通过计数（轮询触发+脚本）

**功能**：检测物品通过光路，自动计数

#### Web界面配置步骤

**步骤1：创建规则**

1. 点击左侧菜单 **外设配置** → 切换到 **外设执行管理** 标签
2. 点击 **<i class="fas fa-plus"></i> 新增规则** 按钮
3. 填写基础配置：
   - **规则名称**：`通过计数`
   - **上报数据**：✅ 启用
   - **启用**：✅ 启用

**步骤2：配置触发器（轮询触发）**

1. 点击 **添加触发** 按钮
2. 填写触发器配置：

   | 字段 | 填写内容 | 说明 |
   |------|---------|------|
   | **触发类型** | 选择 **轮询触发** | 定时检测条件 |
   | **目标外设** | 选择 `photoelectric_01` | 对射光电传感器 |
   | **轮询间隔** | `50` | 50ms（快速检测） |
   | **条件表达式** | `value == 0` | 低电平=光路被遮挡 |

**步骤3：配置动作（脚本计数）**

1. 点击 **添加动作** 按钮
2. 填写：
   - **动作类型**：选择 **命令脚本**
   - **动作参数**：
   ```
   var c=getVar('count',0);
   c++;
   setVar('count',c);
   ```

3. 点击 **保存** 按钮

### 应用场景

| 场景 | 说明 | 检测距离 |
|------|------|---------|
| 物品计数 | 传送带物品统计 | 5-30cm |
| 限位检测 | 机械臂位置限制 | 5-30cm |
| 安全防护 | 危险区域入侵检测 | 5-30cm |

## 注意事项

1. **对准**：发射管和接收管必须正对，偏移会导致误判
2. **环境光**：强光环境可能干扰接收器，建议加遮光罩
3. **检测距离**：典型有效距离 5-30cm（模块型）
4. **响应速度**：光电传感器响应快（<1ms），适合高速计数
