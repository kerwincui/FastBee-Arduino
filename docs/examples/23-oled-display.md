# 实验23：OLED显示屏

## 实验概述

通过 I2C 接口驱动 0.96 寸 OLED 显示屏（SSD1306），显示文字、数值和简单图形。FastBee 内置 `LCD` 外设类型（type: 36）支持 SSD1306/SH1106 OLED 显示。

## 硬件接线

| 开发板标识 | GPIO引脚 | 连接设备 |
|-----------|---------|---------|
| SDA | GPIO21 | OLED SDA（数据线） |
| SCL | GPIO22 | OLED SCL（时钟线） |

> I2C 标准接线，SSD1306 默认地址 0x3C。

## FastBee 外设配置

本实验的 Web 操作入口如下：先在“外设配置”创建硬件对象，再在“外设执行”添加采集、控制或显示规则。新增外设时建议先保持禁用，确认接线后再启用。

### 方式1：Web界面配置（推荐）

#### 步骤1：进入外设管理页面

1. 打开浏览器访问 ESP32 IP 地址
2. 登录后点击左侧菜单 **外设配置**

#### 步骤2：添加OLED显示屏

1. 点击 **<i class="fas fa-plus"></i> 新增外设** 按钮
2. 填写配置：

   **基本信息（左列）：**
   
   | 字段 | 填写内容 | 说明 |
   |------|---------|------|
   | **外设ID** | `oled_display` | OLED显示屏 |
   | **名称** | `OLED显示屏` | 显示名称 |
   | **外设类型** | 选择 **LCD显示** (type: 36) | SSD1306 OLED |
   | **启用** | ✅ 开启 | 默认启用 |

   **引脚和参数（右列）：**
   
   | 字段 | 填写内容 | 说明 |
   |------|---------|------|
   | **引脚配置** | `21,22` | SDA=GPIO21, SCL=GPIO22（顺序重要） |
   | **屏幕宽度** | `128` | 像素（0.96寸=128px） |
   | **屏幕高度** | `64` | 像素（32或64） |
   | **接口类型** | **I2C** (2) | I2C接口 |

3. 点击 **保存**

> ⚠️ **注意**：引脚顺序必须是SDA在前，SCL在后。I2C地址默认0x3C

#### 步骤3：验证配置

1. 在外设列表中查看新增的OLED外设
2. 确认状态为 **运行中**

## 外设执行联动

### 场景1：显示文字信息（平台触发）

**功能**：通过云平台下发文字显示到OLED

#### Web界面配置步骤

**步骤1：创建规则**

1. 点击左侧菜单 **外设配置** → 切换到 **外设执行管理** 标签
2. 点击 **<i class="fas fa-plus"></i> 新增规则** 按钮
3. 填写基础配置：
   - **规则名称**：`OLED显示信息`
   - **上报数据**：✅ 启用
   - **启用**：✅ 启用

**步骤2：配置触发器**

1. 点击 **添加触发** 按钮
2. 填写触发器配置：

   | 字段 | 填写内容 | 说明 |
   |------|---------|------|
   | **触发类型** | 选择 **平台触发** | 接收云平台指令 |
   | **操作符** | 选择 **设置模式** | 接收数据作为动作参数 |

**步骤3：配置动作**

1. 点击 **添加动作** 按钮
2. 填写：
   - **动作类型**：选择 **显示文字**
   - **目标外设**：选择 `oled_display`
   - **第1行**：`FastBee IoT`
   - **第2行**：`温度: 25.3°C`

3. 点击 **保存** 按钮

### 场景2：定时显示传感器数据（定时触发）

**功能**：每3秒在OLED上显示温湿度和IP地址

#### Web界面配置步骤

1. 创建规则，名称：`OLED显示温湿度`
2. 触发器配置：
   - **触发类型**：选择 **定时触发**
   - **定时模式**：选择 **固定间隔**
   - **间隔时间**：`3000`（3秒）
3. 动作配置：
   - **动作类型**：选择 **显示文字**
   - **目标外设**：选择 `oled_display`
   - **第1行**：`T: ${dht_01.temperature}C`
   - **第2行**：`H: ${dht_01.humidity}%`
   - **第3行**：`IP: ${system.ip}`
4. 点击 **保存**

### OLED显示说明

| 参数 | 值 | 说明 |
|------|-----|------|
| 屏幕尺寸 | 0.96寸 | 常用尺寸 |
| 分辨率 | 128x64 | 像素 |
| 显示行数 | 约4行 | 16px字体 |
| 接口类型 | I2C | 地址0x3C |
| 刷新建议 | 1-3秒 | 避免频繁刷新 |

## 注意事项

1. **I2C 地址**：SSD1306 默认 0x3C，部分模块可能为 0x3D
2. **引脚顺序**：pins[0]=SDA, pins[1]=SCL
3. **显示行数**：128x64 屏幕约可显示 4 行文字（16px 字体）
4. **中文支持**：当前版本不支持中文显示，仅支持 ASCII 字符
5. **刷新频率**：建议 1-3 秒刷新一次，频繁刷新影响性能
6. **烧屏**：OLED 长时间显示固定内容会导致烧屏，建议定期关闭或变换内容
