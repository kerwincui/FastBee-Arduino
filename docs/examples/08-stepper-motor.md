# 实验8：步进电机控制

## 实验概述

通过 ULN2003 驱动板控制 28BYJ-48 步进电机，实现精确的角度和转速控制。FastBee 内置 `STEPPER_MOTOR` 外设类型（type: 42），支持半步驱动模式。

## 硬件接线

| 开发板标识 | GPIO引脚 | 连接设备 |
|-----------|---------|---------|
| IN1 | GPIO15 | ULN2003 驱动板 IN1 |
| IN2 | GPIO2 | ULN2003 驱动板 IN2 |
| IN3 | GPIO0 | ULN2003 驱动板 IN3 |
| IN4 | GPIO4 | ULN2003 驱动板 IN4 |

> ULN2003 模块需外接 5V 电源，28BYJ-48 步进电机每圈 2048 步（半步模式）。

## FastBee 外设配置

本实验的 Web 操作入口如下：先在“外设配置”创建硬件对象，再在“外设执行”添加采集、控制或显示规则。新增外设时建议先保持禁用，确认接线后再启用。

### 方式1：Web界面配置（推荐）

#### 步骤1：进入外设管理页面

1. 打开浏览器访问 ESP32 IP 地址
2. 登录后点击左侧菜单 **外设配置**

#### 步骤2：添加步进电机外设

1. 点击 **<i class="fas fa-plus"></i> 新增外设** 按钮
2. 填写配置：

   **基本信息（左列）：**
   
   | 字段 | 填写内容 | 说明 |
   |------|---------|------|
   | **外设ID** | `stepper_01` | 留空自动生成 |
   | **名称** | `ULN2003步进电机` | 显示名称 |
   | **外设类型** | 选择 **步进电机** (type: 42) | 专用步进电机类型 |
   | **启用** | ✅ 开启 | 默认启用 |

   **引脚和参数（右列）：**
   
   | 字段 | 填写内容 | 说明 |
   |------|---------|------|
   | **引脚配置** | `15,2,0,4` | IN1,IN2,IN3,IN4顺序非常重要！ |
   | **每圈步数** | `2048` | 28BYJ-48半步模式=2048步/圈 |
   | **默认转速** | `8` | 8 RPM（建议1-15 RPM） |

3. 点击 **保存**

> ⚠️ **重要**：引脚顺序必须正确（IN1→IN4），否则电机会震动但不转动

#### 步骤3：验证配置

1. 在外设列表中查看新增的步进电机
2. 确认状态为 **运行中**

### 方式2：JSON配置文件

## JSON 配置示例

```json
{
  "peripherals": [
    {
      "id": "stepper_01",
      "name": "ULN2003步进电机",
      "type": 42,
      "enabled": false,
      "pins": [15, 2, 0, 4],
      "params": {
        "stepsPerRevolution": 2048,
        "speed": 8
      }
    }
  ]
}
```

### 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| stepsPerRevolution | 每圈步数（28BYJ-48半步=2048） | 2048 |
| speed | 转速（RPM） | 8 |

## 外设执行联动

### 场景1：正转一圈（平台触发）

**功能**：通过云平台指令控制电机正转一圈（2048步）

#### Web界面配置步骤

**步骤1：创建规则**

1. 点击左侧菜单 **外设配置** → 切换到 **外设执行管理** 标签
2. 点击 **<i class="fas fa-plus"></i> 新增规则** 按钮
3. 填写基础配置：
   - **规则名称**：`步进电机正转`
   - **上报数据**：✅ 启用
   - **启用**：✅ 启用

**步骤2：配置触发器**

1. 点击 **添加触发** 按钮
2. 填写触发器配置：

   | 字段 | 填写内容 | 说明 |
   |------|---------|------|
   | **触发类型** | 选择 **平台触发** | 接收云平台指令 |
   | **操作符** | 选择 **匹配模式** | 匹配指令 |
   | **比较值** | `{"cmd":"forward"}` | 正转指令 |

**步骤3：配置动作**

1. 点击 **添加动作** 按钮
2. 填写动作配置：

   | 字段 | 填写内容 | 说明 |
   |------|---------|------|
   | **动作类型** | 选择 **调用外设** | 调用外设方法 |
   | **目标外设** | 选择 `stepper_01` | 步进电机外设 |
   | **动作参数** | `2048` | 正转2048步（一圈） |

3. 点击 **保存** 按钮

**执行流程**：
```
收到正转指令
  ↓
调用 stepper_01 的 move 方法
  ↓
正转2048步（一圈）
  ↓
动作完成（阻塞执行）
```

> ⚠️ **注意**：stepper_move 动作会阻塞当前规则直到电机转动完成

### 场景2：反转半圈（平台触发）

**功能**：通过云平台指令控制电机反转半圈（-1024步）

#### Web界面配置步骤

**步骤1：创建规则**

1. 点击 **<i class="fas fa-plus"></i> 新增规则**
2. 填写基础配置：
   - **规则名称**：`步进电机反转`
   - **上报数据**：✅ 启用
   - **启用**：✅ 启用

**步骤2：配置触发器**

1. 点击 **添加触发** 按钮
2. 填写：
   - **触发类型**：选择 **平台触发**
   - **操作符**：选择 **匹配模式**
   - **比较值**：`{"cmd":"reverse"}`

**步骤3：配置动作**

1. 点击 **添加动作** 按钮
2. 填写：
   - **动作类型**：选择 **调用外设**
   - **目标外设**：选择 `stepper_01`
   - **动作参数**：`-1024`（负数表示反转，1024步=半圈）

3. 点击 **保存** 按钮

### 步数与角度换算

| 步数 | 角度 | 说明 |
|------|------|------|
| 2048 | 360° | 一整圈 |
| 1024 | 180° | 半圈 |
| 512 | 90° | 四分之一圈 |
| 256 | 45° | 八分之一圈 |
| -2048 | -360° | 反转一整圈 |

## 注意事项

1. **电源**：步进电机需 5V 独立供电，ULN2003 不能从 ESP32 3.3V 取电
2. **发热**：步进电机静止时仍有保持电流，长时间不用应断电
3. **速度限制**：28BYJ-48 最高约 15RPM，超过会丢步
4. **引脚顺序**：IN1-IN4 顺序必须正确，否则电机会震动但不转
5. **阻塞执行**：stepper_move 动作会阻塞当前规则直到完成
