# 光敏自动灯控场景

## 场景描述

利用光敏传感器（ADC）检测环境光照强度，当光照低于阈值时自动开灯（继电器），光照恢复时自动关灯。

## 所需外设

| 外设 | 类型 | 说明 |
|------|------|------|
| light_sensor | GPIO_ANALOG_INPUT(15) | 光敏电阻 ADC |
| relay_light | GPIO_DIGITAL_OUTPUT(12) | 灯继电器 |

## 完整配置流程

### 方式1：Web界面配置（推荐）

#### 第一步：配置外设

**步骤1：进入外设管理页面**

1. 打开浏览器访问 ESP32 IP 地址
2. 登录后点击左侧菜单 **外设配置**

**步骤2：添加光敏传感器**

1. 点击 **<i class="fas fa-plus"></i> 新增外设** 按钮
2. 填写配置：

   | 字段 | 填写内容 | 说明 |
   |------|---------|------|
   | **外设ID** | `light_sensor` | 光照传感器 |
   | **名称** | `光照传感器` | 显示名称 |
   | **外设类型** | **GPIO模拟输入** (type: 15) | ADC采集 |
   | **引脚配置** | `36` | AO对应GPIO36（ADC1） |
   | **初始状态** | `0` | 默认 |

3. 点击 **保存**

**步骤3：添加照明继电器**

1. 再次点击 **<i class="fas fa-plus"></i> 新增外设** 按钮
2. 填写配置：

   | 字段 | 填写内容 | 说明 |
   |------|---------|------|
   | **外设ID** | `relay_light` | 照明继电器 |
   | **名称** | `照明继电器` | 显示名称 |
   | **外设类型** | **GPIO数字输出** (type: 12) | 继电器控制 |
   | **引脚配置** | `15` | IN对应GPIO15 |
   | **初始状态** | `0` | 默认关闭 |

3. 点击 **保存**

---

#### 第二步：配置外设执行规则

**规则1：光照不足开灯**

1. 切换到 **外设执行管理** 标签
2. 点击 **<i class="fas fa-plus"></i> 新增规则** 按钮
3. 填写基础配置：
   - **规则名称**：`光暗开灯`
   - **上报数据**：✅ 启用
   - **启用**：✅ 启用

4. 配置触发器：
   - **触发类型**：选择 **平台触发**
   - **目标外设ID**：填写 `light_sensor`
   - **运算符**：选择 `小于 (<)`
   - **阈值**：`500`（光暗时电压低）

5. 配置动作：
   - **动作类型**：选择 **高电平**
   - **目标外设**：选择 `relay_light`

6. 点击 **保存**

---

**规则2：光照充足关灯**

1. 创建规则，名称：`光亮关灯`
2. 触发器配置：
   - **触发类型**：选择 **平台触发**
   - **目标外设ID**：填写 `light_sensor`
   - **运算符**：选择 `大于 (>)`
   - **阈值**：`2000`（光亮时电压高）

3. 动作配置：
   - **动作类型**：选择 **低电平**
   - **目标外设**：选择 `relay_light`

4. 点击 **保存**

> 💡 **提示**：开灯阈值500，关灯阈值2000，形成1500的迟滞区间，防止频繁切换

## 工作原理

1. 光敏电阻与固定电阻分压，ADC 读取分压点电压
2. 光照越强 → 光敏电阻阻值越小 → ADC 值越高
3. 光照越暗 → 光敏电阻阻值越大 → ADC 值越低
4. 设定暗阈值(500)和亮阈值(2000)形成迟滞区间，避免频繁切换

## 注意事项

1. **迟滞设计**：开灯阈值(500)和关灯阈值(2000)之间有间隔，防止抖动
2. **传感器方向**：部分光敏模块逻辑相反（越暗值越大），需调换比较方向
3. **ADC引脚**：使用 ADC1 通道（GPIO32-39），避免 WiFi 冲突
