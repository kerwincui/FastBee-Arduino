# 用户手册

> 完整的操作指南，涵盖网络配置、外设管理、规则配置、传感器接入和故障排除。

## 1. 版本选择

首先根据您的硬件选择正确的固件版本。详见 [版本选择](./edition-comparison.md) 中的完整功能对比矩阵和构建环境映射表。

**快速建议**：
- ESP32 (4MB)：使用 `esp32-F4R0` (Standard)
- ESP32-S3 (8MB)：使用 `esp32s3-F8R4` (Full)
- ESP32-C3/C6 (4MB)：使用对应 Lite 版本

## 2. 登录设备

### 初始连接

1. 设备上电后自动创建 AP 热点：`FastBee-XXXX`
2. 密码：`fastbee123`
3. 浏览器访问 `http://192.168.4.1`

### 默认账号

| 用户名 | 密码 | 角色 |
|--------|------|------|
| `admin` | `admin123` | 管理员 |

> 首次登录后建议立即修改密码。Full 版本支持多用户管理。

## 3. 网络配置

进入 **网络配置** 页面，选择网络模式：

### WiFi STA 模式（连接路由器）

1. 选择「WiFi STA」模式
2. 点击「扫描」搜索附近 WiFi
3. 选择您的路由器，输入密码
4. 保存后设备自动连接

### 以太网模式（W5500）

1. 选择「以太网」模式
2. 配置 SPI 引脚（默认：MISO=19, MOSI=23, SCK=18, CS=15）
3. 保存后设备通过 DHCP 获取 IP

### 4G 蜂窝网络（EC801E）

1. 选择「4G」模式
2. 输入运营商 APN（如 `cmnet`）
3. 配置 UART 引脚（默认：TX=40, RX=39, PWR=38）
4. 保存后设备自动拨号

> 仅 Full 版本支持以太网和 4G。网络配置详情请参阅 [版本选择](./edition-comparison.md)。

## 4. MQTT 配置

进入 **协议配置** → **MQTT** 页面：

1. 启用 MQTT
2. 输入服务器地址和端口（默认 1883）
3. 输入客户端 ID、用户名和密码
4. 配置主题模板
5. 点击「测试连接」验证

> MQTT 协议详情请参阅 [协议概览](./protocols/README.md)。

## 5. 外设配置

进入 **外设配置** 页面，添加和管理外设。

### 添加外设步骤

1. 点击「添加外设」
2. 选择外设类型
3. 根据提示配置引脚
4. 填写外设名称和描述
5. 保存

### 外设类型参考

完整的外设类型列表和引脚分配原则请参阅 [外设配置](./peripherals/README.md)。

### 传感器配置

支持的传感器列表和接线图请参阅 [传感器完整指南](./peripherals/sensor-guide-complete.md)。

## 6. 规则配置

进入 **外设执行** 页面，创建自动化规则。

### 触发器类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 定时 | Cron 表达式定时 | 每天 8:00 执行 |
| 事件 | 系统事件触发 | WiFi 连接时执行 |
| MQTT | MQTT 消息触发 | 收到指定主题消息 |
| 条件 | 数据条件触发 | 温度 > 30°C |

完整的触发器类型、动作类型和条件运算符请参阅 [外设执行](./periph-exec/README.md)。

## 7. 场景示例

### 温湿度监测

1. 添加 DHT22 传感器（类型 37，Data=4）
2. 添加规则：每 5 分钟读取并上报传感器数据

### 定时灌溉

1. 添加继电器（类型 12，GPIO=16）
2. 添加规则：每天早上 7:00 打开，7:05 关闭

### 远程控制

1. 配置 MQTT 连接到平台
2. 添加规则：MQTT 触发 → 控制继电器

> 更多场景示例请参阅 [示例教程](./examples/README.md)。

## 8. 配置导入导出

### 导出配置

进入 **系统** → **配置管理**，点击「导出」下载全部配置文件。

### 导入配置

1. 准备配置文件（JSON 格式）
2. 点击「导入」上传
3. 设备自动应用配置并重启必要服务

## 9. 故障排除

### 常见问题

| 问题 | 可能原因 | 解决方法 |
|------|---------|---------|
| 无法连接 AP 热点 | 设备未完成启动 | 等待 10-15 秒后重试 |
| 登录失败 | 密码错误 | 确认默认密码 `admin123` |
| MQTT 连接失败 | 网络不通或服务器地址错误 | 检查网络配置和 MQTT 服务器地址 |
| 传感器无法读取 | 引脚配置错误 | 对照 [外设配置](./peripherals/README.md) 检查引脚 |
| 规则不执行 | 规则未启用或条件不满足 | 检查规则开关和触发条件 |
| 内存不足 | 外设过多或碎片化 | 参考 [资源优化](./resource-tuning.md) |

### 恢复出厂设置

长按设备上的 BOOT 按键 10 秒，或通过 Web 界面 **系统** → **恢复出厂设置**。
