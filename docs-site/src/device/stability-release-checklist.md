# 版本发布检查清单

> 新版本发布前的完整验证清单。

## 1. 编译验证

- [ ] 所有目标芯片环境编译通过
  - [ ] `esp32-F4R0` (Standard)
  - [ ] `esp32-F8R4` (Full)
  - [ ] `esp32c3-F4R0` (Lite)
  - [ ] `esp32c6-F4R0` (Lite)
  - [ ] `esp32s3-F8R0` (Standard+OTA)
  - [ ] `esp32s3-F8R4` (Full)
  - [ ] `esp32s3-F16R8` (Full)
- [ ] 编译无警告（或仅允许已知豁免）
- [ ] Flash/RAM 使用量在安全范围内

## 2. 静态检查

- [ ] `pio check` 通过
- [ ] 前端资源完整性检查通过 (`web-smoke-test.js`)
- [ ] i18n 完整性检查通过 (`validate-i18n.js`)
- [ ] 文档链接检查通过 (`validate-doc-links.js`)

## 3. 单元测试

- [ ] `pio test -e native` 全部通过
- [ ] 新增功能有对应测试覆盖
- [ ] 回归测试通过

## 4. 设备实际测试

- [ ] WiFi AP 模式：可连接、可登录、可配置
- [ ] WiFi STA 模式：可连接路由器、可通过 IP 访问
- [ ] 以太网/4G（Full 版）：可正常连接
- [ ] MQTT 连接/断开/重连
- [ ] 外设添加/编辑/删除
- [ ] 外设执行规则（定时/MQTT/条件/事件）
- [ ] OTA 升级（Full 版）
- [ ] 配置导入导出
- [ ] 长时间运行稳定性（≥ 2 小时无重启）

## 5. 关键指标

| 指标 | 阈值 | 状态 |
|------|------|------|
| 空闲堆内存 | > 20KB (no PSRAM) | [ ] |
| 碎片率 | < 30% | [ ] |
| Longest Free Block | > 12KB | [ ] |
| TCP 连接数（稳态） | ≤ 预算 | [ ] |
| API P95 延迟 | < 2s | [ ] |

## 6. 文档

- [ ] 更新文档中版本号
- [ ] 新增功能文档已添加
- [ ] 变更日志已更新
- [ ] 中英文文档同步

## 7. 发布产物

- [ ] `dist/firmware/` 下所有版本的 `factory.bin` 已生成
- [ ] 在线烧录工具可正常加载固件
- [ ] GitHub Release 页面已创建

> 测试详情请参阅 [测试验证](./testing.md)。
