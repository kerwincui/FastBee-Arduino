# 系统功能概览

> FastBee-Arduino Web 管理界面各功能模块介绍。

## 仪表盘

首页仪表盘显示设备运行状态概览：
- 设备信息（名称、ID、固件版本）
- 网络状态（WiFi/以太网/4G 连接状态、IP 地址、信号强度）
- 内存使用（堆总大小、空闲内存、碎片率、PSRAM 状态）
- 系统运行时间
- MQTT 连接状态

## 设备配置

**设备配置**页面管理设备基本参数：
- 设备名称和 ID
- NTP 时间服务器和时区
- 日志级别设置（DEBUG/INFO/WARN/ERROR）
- 设备重启

## 网络配置

**网络配置**页面管理网络接入：
- WiFi STA（连接路由器）— SSID 扫描/密码/静态IP
- WiFi AP（设备热点）— SSID/密码自定义
- 以太网（W5500）— SPI 引脚配置
- 4G 蜂窝（EC801E）— APN/UART 引脚
- mDNS 域名配置
- DNS 服务器设置

> 网络配置操作步骤请参阅 [用户手册](../user-manual.md)。

## 外设配置

**外设配置**页面管理硬件外设：
- 添加/编辑/删除外设
- 支持的外设类型详见 [外设配置](../peripherals/README.md)
- 实时显示外设状态

## 外设执行

**外设执行**页面配置自动化规则：
- 创建/编辑/启用/禁用规则
- 四种触发方式：定时/事件/MQTT/条件
- 完整的动作列表详见 [外设执行](../periph-exec/README.md)

## 协议配置

**协议配置**页面管理通信协议：
- MQTT — 服务器/端口/认证/主题
- Modbus RTU — 波特率/数据位/校验
- TCP — 端口/最大连接数
- HTTP/CoAP — 配置管理

> 协议详情请参阅 [协议概览](../protocols/README.md)。
